# План CI/CD для AnalogJira

## 1. Что такое CI/CD в этом проекте

- **CI (Continuous Integration)** — автоматическая проверка кода при каждом push в GitHub
- **CD (Continuous Delivery)** — автоматический деплой на сервер после успешных проверок

## 2. Инструменты

| Инструмент | Назначение |
|------------|------------|
| GitHub Actions | Автоматизация CI/CD |
| Jest | Запуск тестов |
| Docker | Контейнеризация приложения |
| Render / Railway | Хостинг для деплоя |
